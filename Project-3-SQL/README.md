# Project 3: SQL Data Analysis

## Objective
Use SQL queries to extract business insights from the e-commerce orders dataset.

## Tools Used
- MySQL Workbench

## Queries Covered
- SELECT, WHERE, ORDER BY, GROUP BY
- Aggregate functions: COUNT, SUM, AVG
- JOINs, VIEWs, HAVING clause
- Date functions and string functions
- Subqueries

## Key Insights
- Total revenue by product category
- Top customers by total spend
- Order count by status
- Monthly revenue trends

## Files
- SQL_Queries.sql — all query scripts
- SQL_Results.xlsx — query outputs and analysis
